Voice-Controlled Smart Assistant

This project involved developing a voice-controlled smart assistant capable of recognizing and responding to spoken commands. Using speech recognition technologies and natural language processing (NLP), the assistant can perform tasks such as setting reminders, controlling smart devices, and retrieving information from the web. The project highlights my skills in Python, APIs, and machine learning, specifically in implementing voice recognition and response systems, and demonstrates practical applications of AI in daily life.

Data Mining and Analytics for Agricultural Insights (SUPERMinDS)

As part of the SUPERMinDS research internship, We worked on developing interactive web tools for data mining and analytics, focusing on USDA crop data, weather patterns, and other related factors. This project involved building a web interface to visualize and analyze large datasets, using Python, JavaScript, and various data visualization libraries. The goal was to uncover trends and insights to assist in optimizing agricultural production and sustainability. This experience enhanced my skills in data analysis, web development, and the practical application of machine learning algorithms.
